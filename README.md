# InventoryFrontend

# Clone the repository
git clone https://github.com/kAtApha/Inventory_frontend.git
cd inventory-tracker

# Install dependencies
npm install

# Run the app
ng serve /
ng serve -o


The app will be available at http://localhost:4200

Note: Ensure the backend API is running at http://localhost:5000. Update the API URL in src/environments/environment.ts if needed.
